# Job Finder

I have built a modern Job Finder Web Application using Next.js, React, Tailwind CSS, and MongoDB.
This platform allows users to search, explore, and apply for jobs with a smooth and responsive UI.
It also includes authentication, dashboards, and SEO-optimized pages for real-world performance.

![image alt](https://github.com/muhammadachar435/Job-Finder/blob/master/Screenshot%20(1568).png?raw=true)


# Features

## Job Search & Filters

Search by job title, skills, or location

Dynamic real-time search

Clean and intuitive UI

## User Authentication

Register & Login with secure MongoDB backend

Protected routes

JWT Session-based authentication (depending on your setup)

## Job Details Page

Full job information: company, salary, skills, location, type

Easy apply button

Share options

## Apply for Jobs

Submit application details

Resume upload (optional depending on your implementation)

Unique application IDs

# Dashboard System

User Dashboard: View applied jobs

Admin Dashboard: Add, edit, and delete job postings

# Performance & SEO

Built with Next.js for fast SSR/SSG

SEO-friendly job detail pages

Improved performance on all devices

# Tech Stack

## Layer	Technologies

Frontend	React, Next.js, Tailwind CSS

Backend	Next.js API Routes

Database	MongoDB / Mongoose

Authentication	JWT / Sessions

UI	Tailwind CSS Components

# Getting Started

## 1 Clone the repository
git clone https://github.com/muhammadachar435/Job-Finder.git

## 2 Install dependencies
npm install

## 3 Add your MongoDB URI

Create a .env.local file:

MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_secret_key

## 4 Run the project
npm run dev
